# Docker Task 1: Run Your First Container

An introductory hands-on task demonstrating how to pull, run, verify, and clean up an Nginx container using Docker.

## Objectives Completed
* Pulled the official Nginx image.
* Ran the container in detached mode.
* Mapped port `8080` of the host to port `80` of the container.
* Verified the running server via browser.
* Stopped and removed the container safely.
